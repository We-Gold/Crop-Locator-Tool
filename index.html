Crop Locator Tool

Show Usage Guide

Overview: This tool locates crops from tiffs, given the original image and the crop.

Be Aware: If you notice the tool is slow, try reloading the page. Also, the first scan may be slower than the others

Usage: Upload the original image and the crop. The crop will be highlighted in the original image once the algorithm complete.
The location of the image will be displayed, and additional tools and information will be shown underneath.
As of now, the tool supports normal and rotated crops, but it is in the plan to add support for resliced (different axes) crops.
